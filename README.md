# Dyblocked

A premium, classroom-safe arcade experience that streams 430+ unblocked games with zero installs.

## Running locally
1. Start a static server from the project root:
   ```bash
   python -m http.server 8000
   ```
2. Open [http://localhost:8000](http://localhost:8000) in your browser to browse and launch games.

## Features
- AAA-inspired hero experience with live activity ticker and quick-jump chips.
- Search, category, tag filters, featured-only and shuffle modes.
- Dynamic featured spotlight grid and an immersive modal with instant embeds.
- Responsive design, glassmorphism styling, and smooth scrolling interactions.
